# AgendaPro v0.6.4.0 — Production SaaS Polish

## Objetivo

Polir a experiencia de producao do AgendaPro para venda, portfolio e demonstracao sem mexer em env, runtime, Vercel, Supabase critico ou estrutura de banco.

## Entregas

- Score de prontidao de producao no dashboard da empresa.
- Checklist operacional com botoes reais para corrigir cada ponto.
- Acoes rapidas para proximo ajuste, link publico e Central Dev.
- Botoes do topo do dashboard deixam de ser apenas visuais:
  - `Todas as unidades` abre a aba de unidades.
  - Nome do negocio abre configuracoes.
- CTA da previa do criador deixou de ser botao sem acao e virou elemento visual de preview.
- Versao atualizada para `0.6.4.0`.

## Itens cobertos pelo checklist

- Conta e acesso.
- Plano comercial.
- Pagina publica.
- Servicos e equipe.
- Horarios.
- Comunicacao.
- Relatorios.

## Preservacoes

- Node 20.x preservado.
- Backend permanece CommonJS.
- Nenhuma variavel de ambiente alterada.
- Nenhum SQL novo criado.
- Nenhum fluxo Vercel alterado.
- Nenhum arquivo sensivel incluido.
- `package-lock` continua usando registry publico.
